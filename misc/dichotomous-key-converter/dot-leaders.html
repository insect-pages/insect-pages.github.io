HTML generator for dichotomous key

by J. van der Linden

April 15, 2026

Overview

In this program, I mocked up the raw text of a fake dichotomous key that is formatted similarly to a real key in DOCX format that a colleague created. A Javascript program accepts the raw text of the key (which in this case is shown further down on this page) and spits out a more internet-friendly version, complete with dot leaders (..........) and hyperlinks to couplets further down in the key. As long as the raw text being inputted has consistent basic formatting (such as " ... " for all the dot leaders), this program could in theory be used to automatically convert lengthy dichotomous key DOCX documents to internet-friendlier HTML (with hopefully minor adjustments as needed).

Accessory files:

  • Javascript (.js) file
  • CSS (.css) file
